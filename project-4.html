Neo Pilot
V 1.0

设计亮点

科技感 + 陪伴感，柔和色调塑造亲和力

V1.0 以白色柔和配色与标志性造型细节为核心，打造兼具专业度与亲和力的品牌符号；四视角完整交付（Front / Back / Left / Right），配合像素风表情包与多动态姿态，适配多场景运营与物料落地。

完整衍生体系，从 IP 到周边一步落地

交付像素风表情包（6 款情绪）、场景插画（智行致远 / 星际智航）及徽章、贴纸、帆布袋、保温杯、钥匙扣五类周边设计稿，构建 IP 从视觉到实体的完整衍生链路。

Neo Pilot
V 2.0

设计亮点

全新宇航风潮流造型，升级互动感

2.0 告别初代基础款，专属耳机 + 麦克风加持，自带流动 buff，线条利落更具记忆点；色彩从清新白蓝进阶为梦幻紫粉渐变 + 霓虹光效，低饱和马卡龙色调搭配发光线条，品牌识别度大幅提升。

情绪表达更丰富，场景应用更专业

全新表情包系列捕捉「发现 Bug / 无边无际 / 好不好嘛 / 妙呀！」等真实卖家情绪场景；场景插画（携手 AI 共赢新局 / 专属对接全程护航 / 云端智助全程护航）覆盖品牌推广与运营全触点。
